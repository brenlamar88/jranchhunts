# Site photography

The site loads photos from this folder by filename. The `.jpg` files currently
here are labeled stand-ins — replace each one with the real photo of the same
name (keep the exact filename) and the site picks it up automatically.

| File | Photo to drop in | Used on |
|---|---|---|
| `fallow-buck-1.jpg` | Fallow buck in velvet, head-on in open pasture | Exotics hero, gallery, home strip |
| `fallow-buck-2.jpg` | Fallow buck in velvet, profile in morning field | Home hero, gallery |
| `fallow-buck-3.jpg` | Fallow buck (tag 160) with bachelor group | Home "Exotics" card, gallery |
| `axis-group-1.jpg` | Axis buck and does at the thistle fence line | Gallery hero, exotics CTA, home strip |
| `axis-buck-1.jpg` | Hard-horned axis buck at the tree line | Exotics split image, gallery |
| `photo-6.jpg` … `photo-10.jpg` | Reserved for the next 5 photos | Gallery grid |

Landscape orientation works best; ~2000px wide keeps pages fast.
